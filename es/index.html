Bienvenidos a Mapserver¶

MapServer es una plataforma de Código Abierto para la publicación de datos espaciales y aplicaciones cartográficas interactivas para la web. Originalmente desarrollado a mediados de los 90’s en la Universidad de Minnesota, MapServer es publicado bajo una licencia tipo MIT, y funciona en los principales sistemas operativos (Windows, Linux, Mac OS X).

El Comité Directivo del Proyecto (PSC) MapServer, avalado por OSGeo, gestiona y administra el proyecto, el cual es mantenido, mejorado y apoyado por desarrolladores de todo el mundo. MapServer no es un sistema con todas las funcionalidades de un sistema SIG, ni tampoco aspira a serlo.

Anuncios¶

2014-01-02 - MapServer 6.4.1, 6.2.2, 6.0.4 and 5.6.9 are released

Releases for the stable branches of MapServer have been released in order to fix a potential leakage of information through an SQL injection when using TIME filtering in conjunction with postgis backends. More information can be found in the dedicated issue: #4834.

Changelogs for each specific version can be found here:

  • 6.4.0 to 6.4.1 changes
  • 6.2.1 to 6.2.2 changes
  • 6.0.3 to 6.0.4 changes
  • 5.6.9 contains only the fix for #4834

Head to Download to obtain copies.
