Data Sources¶

Author:Thomas Bonfort
Contact:tbonfort at terriscope.fr

注解

This page is a work in progress

MapCache uses the concept of a “source” as a service that is able to return image data given a set of parameters (namely an extent, an image size, and a projection). Typically, a source is the third party WMS server that you want to put a tilecache infront of.
