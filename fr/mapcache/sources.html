Sources de données¶

Author:Thomas Bonfort
Contact:tbonfort at terriscope.fr

Note

Cette page est un travail en cours

MapCache utilise le concept de “source” pour un service capable de renvoyer des données image à partir d’un jeu de paramètres (principalement une étendue géographique, une taille d’image et une projection). Une source est donc un serveur WMS distant que vous voulez mettre en cache.
