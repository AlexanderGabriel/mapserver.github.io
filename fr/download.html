Binaires¶

Windows¶

MS4W
MapServer pour Windows sur MapTools.org. Si vous êtes un débutant recherchant d’une solution complète MapServer sur les plateformes Windows ou un utilisateur avancé qui veut éviter de compiler sa propre version, c’est ce que vous cherchez. MS4W est un paquet Serveur web/MapServer/MapScript complet. MS4W inclue plusieurs extensions de MapScript (PHP, C#, Python, Java), et dlls supplémentaires pour la gestion de SDE et Oracle. Plusieurs applications web populaires, également disponibles en téléchargement, sont pré-configurés pour MS4W.
OSGeo4W
OSGeo4W est une distribution de binaire d’un large spectre de logiciels Open Source geospatiales pour les environnements Win32 (Windows XP, Vista, etc). OSGeo4W inclue GDAL/OGR, GRASS, MapServer, OpenEV, uDig, ainsi que plusieurs autres paquets (à peu près 70 à l’automme 2008)
FWTools
A collection of Open Source GIS tools provided by Frank Warmerdam. This goes beyond your basic web mapping tools and include many useful utilities for creating, manipulating, and serving geo-spatial data. New versions will probably not appear.
Gisinternals
Une collection de paquest binaires compilés tous les jours, basée sur Mapserver et GDAL SVN (version de développement et stable)

Linux¶

DebianGIS
DebianGIS fournit un certain nombre de paquets incluant MapServer, PostGIS, GDAL, QGIS, et GEOS. Il n’est parfois pas à jour des toutes dernières versions mais DebianGIS propose un environnement d’intégration solide de logiciels SIG Open Source.
Enterprise Linux GIS
Le dépôt Enterprise Linux GIS (ELGIS) fournit des paquets mis à jour de Mapserver et d’autres outils liés à l’OSGEO pour Enterprise Linux et ses dérivés, que sont Red Hat Enterprise Linux (RHEL), CentOS et Scientific Linux.
FGS
L’installateur FGS de Linux est un fichier auto extractible qui installera MapServer avec PHP/MapScript et toutes ses dépendances sur votre système Linux. Il propose un environnement indépendant avec tous les logiciels nécessaires (incluant Apache et PHP) pour lancer des applications de cartographie en PHP/MapScript. La version incluse d’Apache peut être configurée pour fonctionner sur n’importe quel port pour ne pas interférer avec des installations précédentes d’Apache ou d’autres serveurs web sur votre système.
OpenSUSE
Le dépôt Application:Geo d’OpenSUSE fournit des paquets stables pour MapServer et les autres outils OSGeo. Les versions de développement sont aussi mises à disposition pour être testées dans d’autres OBS (Open Build Service) avant d’être migrées dans le dépôt stable.
UbuntuGIS
UbuntuGIS fournit des paquets à jour de Mapserver et d’autres outils liés à l’OSGEO pour les trois dernières versions d’Ubuntu.

Mac OS X¶

Kyng Chaos
William Kyngesburye fournit d’excellents paquets à jour pour Mac OS X. La plupart de la pile SIG Open Source est fournie, incluant GEOS, GDAL, QGIS, et PostGIS.

Application de démo¶

Note

L’application de démonstration n’a pas encore été portée sous MapServer 6

Download MapServer 5.4.x Demo
Téléchargez et décompressez ce fichier sur votre ordinateur. Suivez les explications du fichier ReadMe.txt. Vous devrez probablement déplacer l’arborescence des répertoires de démo dans un endroit approprié de votre serveur httpd. Vous avez également besoin d’éditer certain paramètres de fichiers mapfile et modèles pour refléter les modifications dans les chemins (localisation des répertoires).
