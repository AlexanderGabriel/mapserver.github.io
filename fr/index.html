Bienvenue sur MapServer¶

MapServer est une plate-forme Open Source pour mettre en ligne des données spatiales et des applications cartographiques interactives sur le web. Initialement développé au milieu des années 90 à l’université du Minnesota, MapServer est soumis à la licence MIT-style license, et fonctionne sur la plupart des systèmes d’exploitation (Windows, Linux, Mac OS X). MapServer n’est pas un `SIG`_ au sens propre et n’a pas vocation à le devenir.

Le Comité de Gouvernance de Projet (Project Steering Committee PSC) de Mapserver, approuvé par OSGeo, manage et administre le projet qui est maintenu, amélioré et supporté par des développeurs répartis dans l’ensemble du monde. Voir la page des activités communautaires pour les listes de diffusion, l’irc, etc.

02-01-2014- MapServer 6.4.1, 6.2.2, 6.0.4 et 5.6.9 ont été publiés

Des nouvelles publications pour les branches stables de MapServer ont été effectuées afin de corriger une possible fuite d’information via une injection SQL quand le filtrage TIME est utilisé en conjonction avec le back-office PostGIS. Plus d’informations peuvent être trouvées dans dans le suivi de bug dédié: #4834.

Les logs de changements pour chaque version spécifique peuvent être trouvés ici:

  • 6.4.0 to 6.4.1 changes
  • changements de la version 6.2.1 à 6.2.2
  • changements de la version 6.0.3 à 6.0.4
  • La version 5.6.9 contient seulement la correction pour la le bug #4834

Rendez-vous au téléchargement pour obtenir une copie.
