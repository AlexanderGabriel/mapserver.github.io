Installation de TinyOWS¶

Prérequis¶

TinyOWS need following libraries/applications:
  • LibXML2 (version 2.8 ou plus récente)
  • PostGIS (version 1.5.x ou plus récente)
  • PostgreSQL (avec les en-têtes libpq)
  • Un serveur Web avec le support cgi-bin
  • Le mode Fast-CGI est recommandé

Installez depuis une version stable des sources¶

Un exemple typique d’installation avec téléchargement puis configure, make, make install:

$ wget http://download.osgeo.org/mapserver/tinyows-1.1.0.tar.bz2
$ tar xvjf tinyows-1.1.0.tar.bz2
$ cd mapserver-tinyows
$ ./configure
$ make
$ sudo make install
$ sudo make install-demo

Ensuite, copiez le binaire tinyows dans votre répertoire cgi-bin.

Installez la version courante (dite trunk) depuis Git¶

Pour compiler depuis GitHub, vous devez en premier installer l’utilitaire autoconf et l’application Git

$ git clone git://github.com/mapserver/tinyows.git
$ cd tinyows
$ autoconf
$ ./configure
$ make
$ sudo make install
$ sudo make install-demo

Ensuite, copiez le binaire tinyows dans votre répertoire cgi-bin.
