<h1>Purpose</h1> <p>The main purpose of this cfc is to make mapscript easier to access for Coldfusion developers. This CFC has been written with CF7 but should work equally well with any version from 6 up. It has not been tested using the BlueDragon server.</p>

<h1>Download</h1> <p>The mapscript component can be downloaded from <a href=”http://www.sibern.com.au/index.cfm/spid/1_8.cfm“>Sibern Solutions</a> you should be able to get it directly from <a href=”http://www.sibern.com.au/index.cfm/spid/1_8.cfm?paction=doc.download&amp;document_id=22&amp;category_id=3&amp;in_browser=0“>here</a></p>

<h1>Quick start</h1>

<p>To use mapserver/mapscript you will need to the following </p><ul>

<li>Add the CFC it to a component mapping in Coldfusion</li> <li>Place the mapscript.jar in the classpath or modify the classpath to find the jar file. The simplest method is to put the jar in [CF root]/lib where the standard jar files are. However I would recommend that you place it in its own directory and modify the classpath. Please ensure that the mapscript.jar is compiled with the same version as coldfusion (1.4.2)<br /></li> <li>Modify the library search path to find the mapscript shared library (.so or .dll) or place the shared library in [CF root]/lib </li><li>Restart Coldfusion</li>

</ul>

<p>Depending on the OS and which version of Coldfusion you are running, the installation will be different. You can email me at “simon [at] sibern.com.au” if you are stuck.</p>

<p>Have a read through the cfc and you will see that the different functions are using hints to descibe the functions and parameters.</p>

<p>To get you up and running fast you can use the following as an example of some cfscript.<br /><br />

msapi = createObject(“component”,”mapscript”);<br /> msapi.init(savePath=”/your/directory/to/save/to/”, mapPath=”/where/the/mapfiles/are/”,<br />

mapFile=”mapfilename.map”, shapePath=”/the/location/to/the/shapefiles/”);<br />

mapinfo = msapi.createMap(legend=”false”, width=”450”, height=”450”);</p><p>There is a basic working demo called test.cfm which produces a world map with australian states.</p><p><br /></p>
